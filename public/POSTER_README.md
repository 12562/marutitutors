# Maruti Tutors PDF Poster

This directory contains a printable PDF poster for Maruti Tutors that can be used for marketing and distribution purposes.

## Files Included

- `poster.html` - The HTML file containing the poster design
- `maruti-tutors-logo.png` - The company logo
- `mohit-photo.jpg` - The tutor's photo

## How to Generate PDF

### Method 1: Browser Print to PDF (Recommended)
1. Open `poster.html` in your web browser (you can access it at `http://localhost:3000/poster.html` if running the dev server, or open the file directly)
2. Press `Ctrl+P` (or `Cmd+P` on Mac) to open print dialog
3. Select "Save as PDF" or "Print to PDF"
4. Choose the following settings:
   - **Paper Size**: A3
   - **Orientation**: Portrait
   - **Margins**: None or Minimum
   - **Scale**: 100%
   - **Options**: Check "Print backgrounds"
5. Save as `maruti-tutors-poster.pdf`

### Method 2: Online HTML to PDF Converters
You can use free online tools:
- [html2pdf.com](https://html2pdf.com)
- [pdfcrowd.com](https://pdfcrowd.com)
- [browserling.com/html-to-pdf](https://www.browserling.com/tools/html-to-pdf)

Just upload or paste the `poster.html` content and select A3 paper size.

### Method 3: Browser Developer Tools
1. Open `poster.html` in Chrome/Edge
2. Press `F12` to open Developer Tools
3. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
4. Type "Capture full size screenshot" and select it
5. This will save a PNG, but you can convert it to PDF using online tools

## Poster Specifications

- **Format**: A3 (297 × 420 mm / 11.7 × 16.5 inches)
- **Orientation**: Portrait
- **Resolution**: Optimized for 300 DPI printing
- **Colors**: Professional blue and orange theme matching website

## Content Included

1. **Header**: Maruti Tutors logo and main tagline
2. **Introduction**: Company description and value proposition
3. **Tutor Profile**: Mohit Sharma's photo, credentials, and experience
4. **Subjects Offered**: Complete list of tutoring subjects by level
5. **Contact Information**: Website URL and social media links
6. **Call to Action**: Encouragement to book consultation

## Printing Recommendations

- **Paper**: High-quality glossy or matte paper (100-120 gsm)
- **Printer**: Color laser or inkjet printer
- **Copies**: Print multiple copies for distribution
- **Distribution**: Schools, universities, community centers, online forums

## Customization

The HTML file can be easily customized by:
- Editing text content in the HTML
- Modifying colors in the CSS styles
- Adding or removing sections as needed
- Adjusting font sizes for different paper sizes

## Quick Start

1. Start the development server: `npm run dev`
2. Open `http://localhost:3000/poster.html` in your browser
3. Print to PDF using your browser's print function
4. Your poster is ready for printing!
